404game.js

Turn boring 404 pages into playable mini-games.
Drop-in JS library · zero dependencies · < 30KB

⭐ GitHub v0.1.0 📖 Docs MIT
npm install 404game

🐍 Snake NEON

🐦 Flappy RETRO

🧱 Breakout MINIMAL

🦖 Dino DARK

Controls:
  • Snake — Arrow keys (desktop) / Swipe (mobile)
  • Flappy — Space or Click (desktop) / Tap (mobile)
  • Breakout — Arrow keys + Space (desktop) / Swipe + Tap (mobile)
  • Dino — Space (desktop) / Tap (mobile)
